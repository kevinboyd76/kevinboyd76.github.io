Overview

Deep Learning Image Classification

Goal: Classify human cells into five distinct patterns representing S-phase progression using a convolutional neural network (CNN).

  • Developed deep learning pipeline for image segmentation and classification
  • Preprocessed raw microscopy data and generated labeled training sets
  • Built CNN-based models to predict nuclear patterns across timepoints
  • Integrated model into publication and GitHub-hosted analysis toolkit

Nuclear Colocalization Analysis

Goal: Quantify colocalization between nuclear proteins using multi-channel fluorescence microscopy.

  • Designed reproducible image analysis workflow using Python and skimage
  • Calculated Pearson’s and Manders’ correlation coefficients for nuclear ROIs
  • Enabled batch processing across experimental conditions
  • Supported statistical reporting and figure generation for publication

CUT&Tag Analysis Pipeline

Goal: Develop a modular CUT&Tag data analysis pipeline for chromatin profiling in human cells using snakemake. This pipeline uses fastqs (raw sequencing files) as input.

  • Designed reproducible pipeline for read alignment, peak calling, and signal normalization
  • Generated QC plots and coverage tracks for visualization in IGV
  • Created summary statistics for sample performance and antibody enrichment
  • Documented pipeline for lab use and shared as part of reproducible research initiatives

Cut&Tag Analysis Tab
GitHub Repo

CUT&Tag Replicate Peak Integration

Goal: Integrate biological replicates in CUT&Tag experiments to define consensus peaks. This pipeline performs downstream analysis using genome aligned files as input.

  • Merged aligned BAM files across replicates to increase reproducibility
  • Identified reproducible peaks using a peak overlap threshold
  • Generated Euler plots and summary metrics for peak consistency
  • Generate heatmaps to show signal over specific genomic intervals
  • Provided reproducible scripts for future experiments

Replicate Peak Analysis Tab
GitHub Repo

ChEC-seq Analysis

Goal: Build a reproducible pipeline for analyzing ChEC-seq data from S. cerevisiae and S. pombe.

  • Aligned primary and spike-in reads to dual reference genomes
  • Performed spike-in and CPM normalization for cross-sample comparison
  • Generated signal tracks and enrichment profiles across genomic features
  • Developed modular, Snakemake-based workflow for reproducibility
